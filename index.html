Cara Penggunaan

Format QR Code

Buat QR code di qr-code-generator.com. Isi dengan format Nama|Kelas contoh: Budi Santoso|VI-A — atau cukup nama saja.

Scan Via Kamera

Tekan Mulai Scan Kamera lalu arahkan QR code siswa. Memerlukan HTTPS atau localhost — deploy ke Netlify / GitHub Pages gratis.

Upload Foto QR

Tidak ada server? Foto QR code siswa lalu tekan Upload Foto QR Code. Berfungsi langsung dari file lokal tanpa perlu server.
